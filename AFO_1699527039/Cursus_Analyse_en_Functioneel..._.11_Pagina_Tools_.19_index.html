Tools bij AFO

Bij een aantal onderdelen van deze module kun je gebruik maken van tools. Visual Paradigm (community edition) is bijvoorbeeld een veelgebruikte gratis tool om UML-diagrammen mee te maken (bijvoorbeeld use case diagrammen en domeinmodellen) en met Balsamiq Wireframes kun je schermontwerpen maken (college Schermontwerpen). Je bent vrij om zelf een andere tool te gebruiken, zoals draw.io. Deze tools kun je ook weer bij andere vakken gebruiken, zoals bij het project (KBS). Het tentamen van dit vak is op papier, dan gebruik je deze tools dus niet.
